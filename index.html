Brief

This is a MATLAB package that derives 3D Krawtchouk moment invariants. By inputting a 3D image, it can generate shape features of the object within the image.

Key Features

  • Extract discriminative features from any region-of-interest.
  • The extracted features are translation, scaling, and rotation invariant.
  • The formulation is defined on a discrete space thus do not carry any error.
  • Short runtime required to extract low-order features.

Potential Applications

  • Research on the morphological evolution of species.
  • Prediction of ligand binding sites on protein surfaces.
