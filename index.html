Introduction

This is a MatLab library whose main function is generating the shape signature for the three-dimensional (3D) object.

The signature is implemented by the Krawtchouk moment invariants. The main advantage of this method is the invariants are independent on translation, scaling, and rotation, which makes it possible to describe objects consistently.
